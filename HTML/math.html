What is a function

Function, in mathematics, an expression or rule that defines a relationship between one variable which are the independent variable and the dependent variable; this relationship is commonly symbolized as y = f(x)—in other words “f of x”—and y and x are related such that for every x, there is a unique value.

What is linear function

The linear function is a polynomial function that has the equation y=mx+b that always cuts Y at a point and also cuts X at a point.

  • m=Is equal to the slope of the function

  • n=is the ordinate to the function

    • Points of cut

    Y-axis cutoff point=Is the point of the line whose first coordinate is equal to 0

    X-axis cutoff point=is the point on the line that has 0 in the second coordinate. It is decrypted by setting the function equal to 0 and solving the equation obtained.

    EXAMPLE:

     f (x) 7x-5

    Y-axis cut:

     f(0)=-5

    Is the point:

     (0,-5)

    X-axis cut:

    7x-5=0

    7x=5

    X=5/7

    Is the point:

     (5/7,0)
